# Google Maps Data Scraper & Lead Generation Tool (Playwright + Python)

An advanced automated tool designed to extract business leads and contact information directly from Google Maps based on dynamic search queries. It helps businesses and sales teams eliminate hours of manual data collection.

## Features
- **Dynamic Scraping:** Extracts Business Name, Phone Number, Rating, Reviews Count, Google Maps URL, and official Websites.
- **Data Cleaning:** Automatically filters and removes duplicate listings during the extraction process.
- **Professional Output:** Exports and organizes all structured data directly into a clean Excel sheet (`.xlsx`) ready for marketing campaigns.
